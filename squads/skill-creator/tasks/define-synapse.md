# Task: Define Synapse

**Responsável:** `@skill-architect`
**Gatilho:** `*design`

## Objetivo
Estruturar o DNA da Skill: a Synapse. Isso engloba seu propósito principal e a "Inteligência de Execução", traduzindo os achados de mercado em um fluxo comportamental determinístico e claro para os agentes.

## Instruções de Execução

1. **Consumo de Pesquisa:** Analise os dados entregues pelo `@skill-researcher` na etapa de *Web Research*.
2. **Definição de Fronteiras:** Delimite exatamente qual é o escopo e qual NÃO é o escopo da Skill.
3. **Draft da Synapse:**
   - **Comportamento Core:** Descreva como a Skill atuará (e.g., "Atuar como um avaliador crítico de código", "Ser um gerador de scaffolds").
   - **Fluxo Lógico (Inteligência de Execução):** Monte o passo a passo exato que o agente deve seguir quando usar esta Skill (regras, heurísticas de decisão, outputs esperados).
4. **Validação Humana:** Submeta a Synapse ao usuário para aprovação.
   - *Se recusada:* Ajuste conforme o feedback do usuário.
   - *Se aprovada:* Siga para o handoff com o `@mcp-specialist`.
