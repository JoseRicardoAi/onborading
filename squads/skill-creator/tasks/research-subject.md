# Task: Research Subject

**Responsável:** `@skill-researcher`
**Gatilho:** `*research <assunto>`

## Objetivo
Realizar uma busca profunda na internet para levantar os padrões adotados pelo mercado referente ao "assunto" que a Skill vai atender. O objetivo não é copiar informações rasas, mas entender a estratégia, fluxo de trabalho e "inteligência de execução" necessárias.

## Instruções de Execução

1. **Ativação da Busca:** Inicie a navegação web autônoma utilizando termos estratégicos relacionados ao assunto.
   - *Exemplo:* Se o assunto for "Desenvolvimento de APIs RESTful em Go", pesquise por "Golang REST API best practices", "Go API standard project layout", "Golang API patterns".
2. **Coleta de Inteligência:** Identifique:
   - Quais são as etapas ideais deste fluxo de trabalho?
   - Quais ferramentas, bibliotecas ou dependências a indústria recomenda?
   - Quais os erros mais comuns e como a Skill deve instruir os agentes a evitá-los?
3. **Síntese de Dados:** Formate o conteúdo em um relatório focado em "Inteligência de Execução" (diretrizes claras e táticas acionáveis).
4. **Handoff:** Entregue o relatório final diretamente ao `@skill-architect` para a próxima etapa (Synapse Design).
