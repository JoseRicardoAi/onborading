# Story 4.4: Reabrir Cadastro

## Status

Review

## Story

**As a** RH,
**I want** reabrir um cadastro ou gerar novo link,
**so that** o funcionario possa corrigir informacoes quando necessario.

## Acceptance Criteria

1. RH consegue reabrir cadastro concluido.
2. Sistema gera novo link quando solicitado.
3. Link anterior pode ser invalidado.
4. Status volta para estado adequado de pendencia.

## Tasks / Subtasks

- [x] Criar acao administrativa para reabrir cadastro concluido ou revisado
- [x] Invalidar qualquer link ativo anterior no momento da reabertura
- [x] Gerar novo link temporario para o colaborador
- [x] Voltar o status do cadastro para pendencia de informacoes
- [x] Expor a acao na tela de detalhe e na lista administrativa
- [x] Rodar `npm run lint:web`
- [x] Rodar `npm run build:web`
- [x] Rodar `npm run typecheck:web`

## Completion Notes List

- O RH agora pode reabrir cadastros completos ou revisados e compartilhar um novo convite.
- A reabertura limpa o estado de envio/revisao e invalida links ativos anteriores antes de criar o novo.
- `npm run typecheck:web` exigiu uma segunda execucao apos o build por causa da oscilacao de `.next/types` neste ambiente.
